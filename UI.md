# VividPic UI 设计规范

> 本文档从 PRD.md 提取并细化 HomeScreen 与 Workspace 的界面布局、尺寸与配色规范。所有尺寸以 **96 DPI** 为基准，运行时统一乘以 `Theme::Scale`（由系统 DPI 自动计算）。
>
> 维护者：在修改任何面板/对话框/控件的布局常量后，必须同步更新本文档对应章节。

---

## 1. DPI 缩放规则

```cpp
// Application::Initialize() 中自动检测
UINT dpi = GetDpiForSystem(); // fallback: GetDeviceCaps(hdc, LOGPIXELSX)
Theme::Scale = dpi / 96.0f;

// 使用方式
int  pixel = Theme::GetSize(baseValue);   // baseValue 为 96 DPI 下的像素值
float font = Theme::GetFontSize(basePt);  // basePt 为 96 DPI 下的字号
```

| DPI 设置 | Scale | 示例：基准 200px → 实际 |
|---------|-------|----------------------|
| 96 (100%) | 1.00 | 200 px |
| 120 (125%) | 1.25 | 250 px |
| 144 (150%) | 1.50 | 300 px |
| 192 (200%) | 2.00 | 400 px |

**约束**：所有硬编码像素偏移（如 `+20`、`-10`）**必须**通过 `Theme::GetSize()` 传递，禁止在绘制/布局代码中直接使用裸整数。

## 3. HomeScreen 布局规范

### 3.1 窗口尺寸

```
基准宽度  = 1024 px
基准高度  = 768 px
样式      = WS_OVERLAPPEDWINDOW & ~WS_THICKFRAME（固定大小，可最小化/关闭）
位置      = 屏幕居中
```

运行时：`width = 1024 * Theme::Scale`, `height = 768 * Theme::Scale`

## 5. ColorsPanel（颜色面板）规范

位于 Workspace 左侧面板顶部。

### 5.1 布局

```
┌─────────────────────────────┬──────┐
│                             │ Hue  │
│      SV 色域方块             │ 条   │
│      (SVSquareSize=140)      │ (16) │

├─────────────────────────────┴──────┤

├────────────────────────────────────┤
│ [历史色1] [历史色2] ... [历史色16] │
└────────────────────────────────────┘
```

### 5.2 常量

| 常量 | 基准值 | 缩放 | 说明 |
|------|--------|------|------|
| `SVSquareSize` | 140 | ✓ | SV 选择方块边长 |
| `HueBarWidth` | 16 | ✓ | 色相条宽度 |
| `HueBarHeight` | 140 | ✓ | 色相条高度（与 SV 方块等高） |
| `ColorInfoHeight` | 28 | ✓ | 当前颜色信息行高度 |
| `SwatchSize` | 16 | ✓ | 历史色块边长 |
| `SwatchSpacing` | 4 | ✓ | 历史色块间距 |
| `SwatchRows` | 2 | — | 历史色块行数 |
| `SwatchCols` | 8 | — | 历史色块列数 |

### 5.3 绘制方式

- **SV 方块**：`CreateDIBSection` 预渲染，仅在色相变化时重建。像素直接写入 BGRA。
- **Hue 条**：`CreateDIBSection` 预渲染，创建一次即可。
- **光标**：SV 方块内画白色圆环 + 黑色内环；Hue 条右侧画白色水平线 + 黑色阴影线。
- **历史色块**：GDI `RoundRect` 绘制，点击时更新当前颜色。

| 底部按钮大小 | 24x24 | ✓ | 底部工具栏按钮 |
| 图标按钮间距 | 4 | ✓ | 底部按钮之间间距 |

### 6.3 交互

- **不透明度滑条**：水平拖动，0~100%，实时更新 `Layer::opacity` 并触发 composite。
- **混合模式下拉**：8 种模式（Normal, Multiply, Screen, Overlay, Difference, Add, Subtract, Darken, Lighten）。
- **复选框**：点击切换 `protectAlpha`、`clipMask` 等布尔属性。
- **图层项点击**：选中当前图层，高亮背景 `HighlightBlue` @ 30% 透明度。
- **可见性图标**：点击切换 `Layer::visible`。
- **底部按钮**：
  - `+`：在当前图层上方新建空白图层
  - `复制`：深拷贝当前图层
  - `↓合并`：将当前图层向下合并（Merge Down）
  - `🗑`：删除当前图层（至少保留 1 个图层）

## 7. NavigatorPanel（导航器面板）规范

位于 Workspace 右侧面板顶部。

### 7.1 布局

```
┌────────────────────────────────────┐
│ 导航器                              │ ← 标题栏
├────────────────────────────────────┤
│                                    │
│      ┌──────────┐                  │
│      │ 缩略图    │                  │
│      │          │                  │
│      │  ┌────┐  │                  │ ← 视图范围框
│      │  │框  │  │                  │
│      │  └────┘  │                  │
│      └──────────┘                  │
│                                    │
├────────────────────────────────────┤
│ 100%  [ - ] [====●====] [ + ]      │ ← 缩放控制
└────────────────────────────────────┘
```

### 7.2 常量

| 常量 | 基准值 | 缩放 | 说明 |
|------|--------|------|------|
| 标题栏高度 | 24 | ✓ | "导航器" 标题区 |
| 缩略图区边距 | 8 | ✓ | 缩略图到面板边缘的内边距 |
| 视图框边框 | 2 | ✓ | 视图范围矩形边框宽度 |
| 视图框颜色 | `#0078D7` | — | `HighlightBlue`，半透明填充 |
| 缩放控制高度 | 24 | ✓ | 底部缩放滑条/按钮行 |
| 缩放按钮宽 | 24 | ✓ | `-` / `+` 按钮宽度 |

### 7.3 绘制与交互

- **缩略图**：从 `LayerManager::CompositeToBuffer()` 的 RGBA buffer 采样，用 `StretchBlt` 缩放到缩略图区。
- **视图框**：
  - 计算：`viewRect = canvasOffset * (thumbSize / canvasSize)`
  - 绘制：半透明蓝色填充 + 实线边框。
- **点击平移**：
  - `LButtonDown`：记录拖拽起点，进入拖拽模式。
  - `MouseMove`：计算鼠标在缩略图上的偏移 → 映射为画布 offset → `CanvasView::SetOffset()` → `Invalidate` 画布 + 导航器。
  - `LButtonUp`：结束拖拽。
