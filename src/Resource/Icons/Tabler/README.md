# Tabler Icon Subset

This folder vendors the MIT-licensed Tabler Icons reference subset used by
CloverPic's core soft-rendered UI.

Runtime rendering does not load these files from the platform adapter. The core
build uses matching `IconId` vector definitions in `Core/Presentation`, while
this folder keeps the source/resource attribution and a compact SVG reference
set for future asset regeneration.

Source project: https://github.com/tabler/tabler-icons
License: MIT, see `LICENSE`.
